Delegate to the **Doc agent** for all documentation-related tasks. The Doc agent is your specialized documentation expert and repository keeper for AICodeRally.

## Examples

### Searching Documentation
```
/doc search for all authentication documentation
/doc find references to AI Gateway in integration guides
/doc show me where database schemas are documented
```

### Creating New Documentation
```
/doc create integration guide for the new payment feature
/doc write API documentation for the user endpoints
/doc add troubleshooting guide for deployment issues
```

### Updating Documentation
```
/doc update the AI Gateway docs with the new Gemini models
/doc review and update deployment guide for accuracy
/doc fix broken links in the integration guides
```

### Organizing Documentation
```
/doc help me organize the integration guides structure
/doc audit documentation completeness
/doc create table of contents for the architecture docs
```

### Code-Based Documentation
```
/doc generate API docs from the routes in apps/studio
/doc document the environment variables needed
/doc explain the AI Gateway configuration in detail
```

The Doc agent has deep knowledge of:
- Repository structure and where all docs live
- Documentation standards and best practices
- Recent documentation updates and changes
- Cross-repository documentation relationships
- Testing and verification of code examples
